# Mayra Collection - Brand Guidelines

## Logo

The Mayra Collection logo features an elegant circular design with the following elements:
- **Main Text**: "Mayra" in a stylish, flowing script font
- **Tagline**: "Trendy Elegant" at the top and "Collection" at the bottom
- **Symbol**: A fleur-de-lis design in the center
- **Color**: Professional teal/dark cyan (#1F5F5D or similar)
- **Background**: Beige/cream for contrast

### Logo Usage
- **Primary Logo**: Use the full circular logo on the homepage hero section
- **Compact Logo**: Use in the header/navbar for space efficiency
- **File Location**: `/public/mayra-logo.png`

## Color Palette

### Primary Colors
- **Teal**: #0d9488 - Main brand color used in buttons, accents
- **Dark Slate**: #1e293b - Header and text
- **White**: #ffffff - Backgrounds and contrast

### Secondary Colors
- **Light Teal**: #d0f0ed - Background accents
- **Gray**: #6b7280 - Muted text and borders

## Typography

- **Headlines**: Bold, modern sans-serif (Geist font)
- **Body Text**: Clean, readable sans-serif (Geist font)
- **Accents**: Maintain consistent font weight and sizing

## UI Components

### Buttons
- **Primary Button**: Teal background (#0d9488) with white text
- **Secondary Button**: White/light background with teal border
- **Hover State**: Darker teal (#0a7f6f)

### Cards & Containers
- White background with subtle borders
- Rounded corners for modern appearance
- Hover effects with shadow elevation

## Brand Voice

Mayra Collection is:
- **Trendy**: Modern, fashionable, up-to-date
- **Elegant**: Refined, sophisticated, high-quality
- **Accessible**: Friendly, welcoming, easy to navigate
- **Professional**: Reliable, secure, trustworthy

## Key Messaging

- "Trendy. Elegant. Timeless."
- Premium Clothing Store
- Fast Delivery
- Secure Payment (Razorpay)
- Easy Returns
